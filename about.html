Hi, nama saya Radi Hartono, seorang mahasiswa jurusan Informatika di Universitas Dehasen Bengkulu. Saya tertarik pada pengembangan web dan saat ini sedang mempelajari HTML, CSS, dan JavaScript.

Selain itu, saya juga tertarik dengan dunia keamanan siber dan sering bereksperimen dengan teknik hacking etis untuk meningkatkan keterampilan pemrograman saya. Saya suka berbagi pengetahuan dan terlibat aktif di komunitas Linux lokal.

Saya percaya bahwa dengan kerja keras dan pembelajaran terus-menerus, saya akan dapat menguasai berbagai teknologi di bidang pengembangan web dan keamanan siber. Jangan ragu untuk menghubungi saya jika ingin berkolaborasi dalam proyek-proyek menarik!
